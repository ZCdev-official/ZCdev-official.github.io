ZCdev

Modern design,
intentional code.

A developer at heart.

ZCdev is the name I (Zachary Cowan) use for developing. The name provides an easily recognizable way for people to identify me as the developer of some code or design. Although code and design are typically regarded as separate, I stress the benefits of treating them as a package.

"Your code should be a reflection of your design, and vice versa."

Design should not just look good; it should function well. If a design is pleasing to the eye but does not serve its purpose, it will ultimately fail. Each part of a design should be able to work well with the code.

Code

Coding is a language for math. With it, developers can do some very creative things. And just like any other language, coding is used to communicate with people.

Design

Although coding is used to communicate with people, it is usually too complicated for anyone to care about it. Graphical designs allow developers to communicate their code in a much simpler way.

Publish

Coding and designing are great. However, nobody cares unless they are able to see the final product. As a result, it is crucial for developers to publish their work so that others may enjoy it.
